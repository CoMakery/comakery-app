IV. Transfers API

Create and cancel transfers, retrieve transfer data.

CREATE

POST /api/v1/projects/:project_id/transfers

Returns created transfer details (See GET for response details)

Parameters

Name Description
project_id project id
transfer[source] transfer source ( earned bought mint burn ), defaults to earned
transfer[amount] transfer amount (same decimals as token)
transfer[quantity] transfer quantity (2 decimals)
transfer[total_amount] transfer total_amount (amount times quantity, same decimals as token)
transfer[account_id] transfer account id
transfer[description] transfer description

Response Fields

Name Description
errors array of errors
