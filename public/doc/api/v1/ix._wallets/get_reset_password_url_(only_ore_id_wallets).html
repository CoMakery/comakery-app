IX. Wallets API

Create, delete and retrieve account wallets.

GET RESET PASSWORD URL (ONLY ORE_ID WALLETS)

POST /api/v1/accounts/:id/wallets/:wallet_id/password_reset

Returns reset password url for wallet

Parameters

Name Description
id account id
wallet_id wallet id
redirect_url url to redirect after password change

Response Fields
