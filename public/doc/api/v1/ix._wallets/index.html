IX. Wallets API

Create, delete and retrieve account wallets.

INDEX

GET /api/v1/accounts/:id/wallets

Returns an array of wallet objects

Parameters

Name Description
id account id
page page number

Response Fields

Name Description
id wallet id
address wallet address
source wallet source ["user_provided", "ore_id"]
state wallet state ["pending", "pending_manual", "unclaimed", "ok", "unlinking"]
blockchain wallet blockchain ["bitcoin", "bitcoin_test", "cardano", "cardano_test", "qtum", "qtum_test", "eos", "eos_test", "tezos", "constellation", "constellation_test", "ethereum", "ethereum_ropsten", "ethereum_kovan", "ethereum_rinkeby", "algorand", "algorand_test", "algorand_beta"]
tokens wallet tokens
provision_tokens wallet tokens which should be provisioned with state for each token
createdAt creation timestamp
updatedAt update timestamp
