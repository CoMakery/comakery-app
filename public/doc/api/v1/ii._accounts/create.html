II. Accounts API

Retrieve and manage account data, balances, interests.

CREATE

POST /api/v1/accounts

Returns created account data (See GET for response details)

Parameters

Name Description
account[managed_account_id] id
account[email] email
account[first_name] first name
account[last_name] last name
account[nickname] nickname
account[image] image
account[country] counry
account[date_of_birth] date of birth (YYYY-MM-DD)
account[projects] list of project ids to subscribe

Response Fields

Name Description
errors array of errors
