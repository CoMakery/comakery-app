VII. Blockchain Transactions API

Generate blockchain transactions for project to process and submit to blockchain.

GENERATE TRANSACTION - NO TRANSFERS

POST /api/v1/projects/:project_id/blockchain_transactions

Returns empty response if no transactables available for transaction

Parameters

Name Description
project_id project id
transaction[source] transaction source wallet address
transaction[nonce] transaction nonce
blockchain_transactable_id transactable id to generate transaction for
blockchain_transactable_type transactable type to generate transaction for â€“ Award (Default), TransferRule, AccountTokenRecord

tx_raw transaction in HEX
status transaction status (created pending cancelled succeed failed)

createdAt transaction creation timestamp
updatedAt transaction update timestamp
