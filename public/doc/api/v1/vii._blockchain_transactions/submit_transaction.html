VII. Blockchain Transactions API

Generate blockchain transactions for project to process and submit to blockchain.

SUBMIT TRANSACTION

PUT /api/v1/projects/:project_id/blockchain_transactions/:id

Marks transaction as pending and returns transaction details, see GENERATE TRANSACTION for response fields

Parameters

Name Description
project_id project id
id transaction id
transaction[tx_hash] transaction hash
