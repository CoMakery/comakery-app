VIII. Reg Groups API

Create and delete reg groups, retrieve reg group data.

GET

GET /api/v1/projects/:project_id/reg_groups/:id

Returns data for a single reg group.

Parameters

Name Description
project_id project id
id reg group id

Response Fields

Name Description

name reg group name
blockchain_id reg group id on blockchain
token_id reg group token id
createdAt creation timestamp
updatedAt update timestamp
