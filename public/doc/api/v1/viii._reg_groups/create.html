VIII. Reg Groups API

Create and delete reg groups, retrieve reg group data.

CREATE

POST /api/v1/projects/:project_id/reg_groups

Returns created reg group details (See GET for response details)

Parameters

Name Description
project_id project id
reg_group[name] reg group name
reg_group[blockchain_id] reg group id on blockchain (will be auto-generated if not provided)

Response Fields
