Comakery Whitelabel API

All of the endpoints below are only accessible via a whitelabel domain, with data scoped to the according instance.

I. General

Details on authentication, caching, throttling, inflection and pagination.

  • AUTHENTICATION
  • AUTHENTICATION – INCORRECT PUBLIC KEY HEADER
  • AUTHENTICATION – INCORRECT PROOF
  • CACHING
  • THROTTLING
  • INFLECTION
  • PAGINATION

II. Accounts

Retrieve and manage account data, balances, interests.

  • GET
  • CREATE
  • CREATE – ERROR
  • UPDATE
  • UPDATE – ERROR
  • INTERESTS
  • CREATE INTEREST
  • CREATE INTEREST – ERROR
  • REMOVE INTEREST
  • VERIFICATIONS
  • CREATE VERIFICATION
  • CREATE VERIFICATION – ERROR
  • TOKEN BALANCES

III. Projects

Retrieve projects data.

  • INDEX
  • GET

IV. Transfers

Create and cancel transfers, retrieve transfer data.

  • INDEX
  • GET
  • CREATE
  • CREATE – ERROR
  • CANCEL
  • CANCEL – ERROR

V. Transfer Rules

Create and delete transfer rules, retrieve transfer rules data.

  • INDEX
  • GET
  • CREATE
  • CREATE – ERROR
  • DELETE

VI. Account Token Records

Create and delete account token records, retrieve account token record data.

  • INDEX
  • GET
  • CREATE
  • CREATE – ERROR
  • DELETE

VII. Blockchain Transactions

Generate blockchain transactions for project to process and submit to blockchain.

  • GENERATE TRANSACTION
  • GENERATE TRANSACTION – WITH PROJECT TRANSACTION API KEY
  • GENERATE TRANSACTION – TRANSACTABLE ID
  • GENERATE TRANSACTION – TRANSACTABLE TYPE
  • GENERATE TRANSACTION - NO TRANSFERS
  • SUBMIT TRANSACTION
  • SUBMIT TRANSACTION – HASH MISMATCH
  • CANCEL TRANSACTION

VIII. Reg Groups

Create and delete reg groups, retrieve reg group data.

  • INDEX
  • GET
  • CREATE
  • CREATE – ERROR
  • DELETE

IX. Wallets

Create, delete and retrieve account wallets.

  • INDEX
  • CREATE WALLET
  • CREATE WALLET – ERROR
  • GET WALLET
  • REMOVE WALLET
  • GET RESET PASSWORD URL (ONLY ORE_ID WALLETS)
